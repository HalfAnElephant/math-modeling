> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修正问题一当前“先尽量满载生成2趟，再分配给K架无人机”的口径，建立时间优先验证，判断 `K=4` 是否能通过主动拆分并行航次降低 `T_u`。

**Architecture:** 新增独立模块 `c_uav_inspection/problem1_time.py`，用子集候选航次预计算加 K 路划分动态规划求解“最多 K 条并行航次覆盖全部目标，使最大航次时长最小”。保留旧满载装箱结果作为基准，不直接删除旧算法。

**Tech Stack:** Python 3.11, standard library `dataclasses`, `math`, existing `c_uav_inspection.data`, `model`, `search`, `problem1`, `objective`.

## 1. 背景与根因

当前问题一结果显示 `K=2,3,4` 的无人机阶段时间都为 `639.33 s`。这不是因为题目证明了 `K=4` 无法更快，而是因为当前流程先把基础巡检任务尽量装成2趟：

```text
nearest_neighbor_order
-> split_order_into_energy_feasible_routes
-> improve_route_by_two_opt
-> _assign_routes_to_uavs
```

该流程偏向“少航次、充分利用单趟电量”。题目要求优先考虑时间时，`K=4` 可能应该主动增加并行航次，用更多往返换取更短最大完成时间。因此必须新增时间优先验证。

## 4. 子集候选航次预计算

- [ ] **Step 3: 实现目标索引工具**

```python
def _target_maps(data: ProblemData) -> tuple[tuple[int, ...], dict[int, float], dict[int, int]]:
    node_ids = tuple(target.node_id for target in data.targets)
    base_hover = {target.node_id: target.base_hover_time_s for target in data.targets}
    node_to_bit = {node_id: idx for idx, node_id in enumerate(node_ids)}
    return node_ids, base_hover, node_to_bit
```

- [ ] **Step 4: 实现 mask 转 node tuple**

```python
def _nodes_from_mask(mask: int, node_ids: tuple[int, ...]) -> tuple[int, ...]:
    return tuple(node_ids[idx] for idx in range(len(node_ids)) if mask & (1 << idx))
```

- [ ] **Step 5: 实现 `precompute_problem1_subset_routes`**

```python
def precompute_problem1_subset_routes(
    data: ProblemData,
    improve: bool = True,
) -> dict[int, SubsetRouteCandidate]:
    node_ids, base_hover, _ = _target_maps(data)
    candidates: dict[int, SubsetRouteCandidate] = {}
    full_mask = (1 << len(node_ids)) - 1

    for mask in range(1, full_mask + 1):
        subset_nodes = _nodes_from_mask(mask, node_ids)
        order = nearest_neighbor_order(data, list(subset_nodes))
        hover_times = {node_id: base_hover[node_id] for node_id in subset_nodes}
        route = UAVRoute(
            uav_id=0,
            sortie_id=1,
            node_sequence=(0,) + order + (0,),
            hover_times_s=hover_times,
        )
        if improve:
            route = improve_route_by_two_opt(route, data)
        metrics = evaluate_uav_route(route, data)
        if metrics.feasible_energy:
            candidates[mask] = SubsetRouteCandidate(
                mask=mask,
                node_ids=subset_nodes,
                route=route,
                duration_s=metrics.duration_s,
                energy_j=metrics.energy_j,
                feasible_energy=True,
            )
    return candidates
```

## 5. K 路时间优先动态规划

- [ ] **Step 6: 实现 `solve_problem1_time_priority_for_k`**

函数签名：

```python

```

入参校验：

```python
if k <= 0:
    raise ValueError(f"k must be positive, got {k}")
```

核心 DP：

```python
node_ids, base_hover, _ = _target_maps(data)
n = len(node_ids)
full_mask = (1 << n) - 1
candidates = precompute_problem1_subset_routes(data, improve=improve)

best: list[dict[int, float]] = [{0: 0.0}]
parent: list[dict[int, tuple[int, int]]] = [{}]

for route_count in range(1, k + 1):
    prev = best[route_count - 1]
    current: dict[int, float] = dict(prev)
    current_parent: dict[int, tuple[int, int]] = {}
    for covered_mask, covered_time in prev.items():
        remaining = full_mask ^ covered_mask
        sub = remaining
        while sub:
            candidate = candidates.get(sub)
            if candidate is not None:
                new_mask = covered_mask | sub
                new_time = max(covered_time, candidate.duration_s)
                old_time = current.get(new_mask, inf)
                if new_time < old_time - 1e-9:
                    current[new_mask] = new_time
                    current_parent[new_mask] = (covered_mask, sub)
            sub = (sub - 1) & remaining
    best.append(current)
    parent.append(current_parent)
```

说明：这里允许“最多 K 条并行航次”。`current` 初始复制 `prev`，表示少用一条航次仍然可行。

- [ ] **Step 7: 重构最优路线**

```python
best_route_count = None
best_time = inf
for route_count in range(1, k + 1):
    value = best[route_count].get(full_mask, inf)
    if value < best_time - 1e-9:
        best_time = value
        best_route_count = route_count

if best_route_count is None or best_time == inf:
    raise InfeasibleError(f"No time-priority partition is feasible for k={k}")

route_masks: list[int] = []
mask = full_mask
route_count = best_route_count
while mask:
    prev_mask, sub = parent[route_count][mask]
    route_masks.append(sub)
    mask = prev_mask
    route_count -= 1
route_masks.reverse()
```

- [ ] **Step 8: 给路线赋无人机编号并汇总**

```python
assigned_routes: list[UAVRoute] = []
for idx, mask in enumerate(route_masks, start=1):
    base_route = candidates[mask].route
    assigned_routes.append(UAVRoute(
        uav_id=idx,
        sortie_id=1,
        node_sequence=base_route.node_sequence,
        hover_times_s=dict(base_route.hover_times_s),
    ))

routes = tuple(assigned_routes)
summary = summarize_uav_solution(routes, data, data.params.battery_swap_time_s)

total_hover_by_node = {node_id: 0.0 for node_id in node_ids}
for route in routes:
    for node_id, seconds in route.hover_times_s.items():
        total_hover_by_node[node_id] += seconds

return TimePriorityProblem1Solution(
    k=k,
    routes=routes,
    total_hover_by_node=total_hover_by_node,
    summary=summary,
    solver_name="time_priority_partition",
    route_count=len(routes),
)
```
