## 5. 修改直接确认评分

- [ ] **Step 6: 修改 `_direct_confirm_score`**

当前返回：

```python
return ground_savings / (1.0 + max(energy_penalty, 0.0) + extra_hover_cost)
```

改为：

```python
priority = max(target.priority_weight, 1)
return (
    priority * ground_savings
    / (1.0 + max(energy_penalty, 0.0) + extra_hover_cost)
)
```

解释：

- `priority_weight=3` 的目标在相同人工节省与额外悬停成本下，评分约为 `priority_weight=1` 的三倍。
- `max(target.priority_weight, 1)` 避免异常数据中的 0 权重使目标永远没有直接确认机会。

- [ ] **Step 8: 修改接受逻辑**

当前逻辑：

```python
accepted = False
if new_closed_time < cur_closed_time - 1e-9:
    accepted = True

```

改为：

```python
accepted = False
if new_closed_time < cur_closed_time - 1e-9:
    accepted = True
elif (
    new_weighted < cur_weighted
    and new_closed_time <= cur_closed_time * 1.03 + 1e-9
):
    accepted = True
elif (
    new_manual < cur_manual
    and new_closed_time <= cur_closed_time * 1.03 + 1e-9
):
    accepted = True
```

这样优先级权重不仅影响候选顺序，也影响候选接受。

def test_direct_confirm_score_multiplies_priority_weight():
    data = load_problem_data(DATA_PATH)
    target = next(t for t in data.targets if t.node_id == 2)
    threshold = effective_direct_threshold(target, 1.0)
    extra_hover_cost = (threshold - target.base_hover_time_s) / max(target.base_hover_time_s, 1.0)
    ground_savings = (
        data.ground_time_s[("P0", target.manual_point_id)]
        + data.ground_time_s[(target.manual_point_id, "P0")]
        + target.manual_service_time_s
    )
    roundtrip_energy = data.flight_energy_j[(0, 2)] + data.flight_energy_j[(2, 0)]
    energy_penalty = roundtrip_energy / data.params.effective_energy_limit_j
    expected = target.priority_weight * ground_savings / (1.0 + energy_penalty + extra_hover_cost)

    assert _direct_confirm_score(data, 2, 1.0) == pytest.approx(expected)

- [ ] **Step 17: 第 4.3 问题2模型新增公式**

在人工复核集合定义后增加：

```markdown
考虑赛题附件给出的目标优先级权重 $p_i$，定义加权人工复核代价为

$$
C_M=\sum_{i\in M}p_i.
$$
```

目标函数改为：

```markdown
$$
\min F_2=
0.45\,\operatorname{norm}(T)
+0.20\,\operatorname{norm}(T_g)
+0.15\,\operatorname{norm}(C_M)
+0.10\,\operatorname{norm}(|M|)
+0.05\,\operatorname{norm}\!\left(\sum_{r\in R}E_r\right)
+0.05\,\operatorname{norm}(\sigma_L).
$$
```

- [ ] **Step 18: 第 5.6 重建搜索公式更新**

评分公式改为：

```markdown
$$
\operatorname{score}_i=
\frac{p_i\operatorname{ground\_saving}_i}
{1+\operatorname{energy\_penalty}_i+\operatorname{extra\_hover\_cost}_i}.
$$
```

接受准则写为：

```markdown
若 $T'<T$，或 $C_M'<C_M$ 且 $T'\le 1.03T$，或 $|M'|<|M|$ 且 $T'\le 1.03T$，则接受候选方案。
```
