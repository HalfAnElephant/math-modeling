# C 题可复现实验执行总览

> **执行规则：** 按本目录文件编号从小到大执行。每个子计划最后的验收命令通过后，才能进入下一份子计划。

**项目根目录：**

```bash
cd "/Users/xcy/Documents/100 项目/数学建模"
```

**总目标：** 建立 C 题“多无人机空中巡检 + 物业复核闭环”的可复现实验流程，输出可行路线、直接确认策略、敏感性分析、图表和论文结果说明。

**核心修正必须始终保留：**

- 多目标函数必须先归一化，再加权求和。
- 悬停时间是可拆分连续资源，允许同一目标跨无人机、跨趟次累计完成。
- 问题 2 的直接确认升级必须使用重建搜索，不使用“超限就单点新建航次”的短视策略。
- 直接确认阈值敏感性必须使用 `max(base_hover_time_s, direct_confirm_time_s * multiplier)`。
- 物业人员必须在全部无人机任务结束后出发，闭环时间为 `T_u + T_g`。

## 子计划顺序

1. `PLAN/01-环境与数据加载.md`
   - 创建包结构。
   - 读取 Excel 数据。
   - 完成确定性数据核验。

2. `PLAN/02-核心模型与归一化目标.md`
   - 实现无人机路线、能耗、时间和负载指标。
   - 实现维度无关的归一化目标函数。

3. `PLAN/03-可拆分悬停与问题1.md`
   - 实现最近邻顺序。
   - 实现可拆分悬停装箱。
   - 实现问题 1 多无人机基础巡检求解器。

4. `PLAN/04-问题2闭环与重建搜索.md`
   - 实现物业复核 TSP。
   - 实现直接确认阈值下界。
   - 实现问题 2 直接确认集合的重建搜索。

5. `PLAN/05-实验图表与论文说明.md`
   - 生成 CSV/JSON 实验结果。
   - 生成 PNG 图表。
   - 写入论文结果说明。

6. `PLAN/06-最终验收.md`
   - 从零重建全部输出。
   - 检查文件清单、关键约束和风险表述。
